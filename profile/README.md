[The Scala Workshop](https://workshop.scala-lang.org), the spiritual successor of the Scala Symposium, focuses on the [Scala programming language](https://scala-lang.org), bringing together researchers and practitioners to discuss its design, implementation, and applications.
Topics include language features, tools, and libraries, as well as functional and object-oriented programming paradigms.
The workshop promotes collaboration between academia and industry to advance Scala's development and usage.
The audience includes researchers, developers, and industry professionals who work with Scala or related technologies.

# About this organisation

As its name highlights, this organisation only focuses on The Scala Workshop.
Most of the repos in it are private, but here is a list of interesting public repositories:
- [workshop.scala-lang.org](https://github.com/scala-workshop/workshop.scala-lang.org) contains the source code of the workshop's website;
- [brand-assets](https://github.com/scala-workshop/brand-assets) contains all the branding assets of the workshop.

For more information about The Scala Programming Language, please visit the [main website](https://scala-lang.org) or the [scala organisation on GitHub](https://github.com/scala).
